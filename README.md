# SPOTIFY CONSOLE CPP

## Project Status
🚧 Currently under development  

A console-based Spotify-inspired music manager developed in C++ as an academic project focused on data structures, file handling, and object-oriented programming.

## FEATURES:
### Authentication
- User login system
- Maximum of 3 login attempts

### Music Library
- Liked Songs playlist
- Personal Mix playlist
- Add songs
- Remove songs
- Search songs
- Display song lists

### Playback System
- Sequential playback
- Random playback
- Song lyrics visualization
- Audio playback support

## TECHNICAL CONCEPTS
This project was built to practice and strengthen core Computer Science concepts:
- Object-Oriented Programming (OOP)
- Doubly Linked List
- Dynamic Memory Management
- File Handling
- Modular Software Design
- Search and Playlist Management Algorithms

## TECHNOLOGIES
- C++
- Standard Template Library (STL)
- File I/O
- Git
- GitHub
